Contribute
==========

All contributions are welcome :

* Feedbacks, bugs
* New use cases
* Bugs fixes
* :doc:`new-provider`

Please use the GitHub issue tracker.

Running the tests
-----------------

.. code-block:: bash

    pip install -r test-requirements
    tox

Running syntax checker
----------------------

.. code-block:: bash

    pip install -r test-requirements
    flake8
