Tutorials¶

This section presents few tutorials. The `EnOS Tutorial on top of Grid’5000`_ shows you how to deploy,configure, stress and collect metrics of OpenStack on top of Grid’5000 using Enos. The `OpenStack SDK`_ shows you how to reads Enos environment file to then interact with OpenStack using the OpenStack SDK to create a new OpenStack project with users in it.
